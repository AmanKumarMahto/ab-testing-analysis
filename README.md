# A/B Testing & Conversion Optimization Analysis

## Objective
Analyze two variants (A vs B) to determine which performs better in terms of conversion rate.

## Dataset
Simulated dataset of 1000+ users with two variants (A and B).

## Approach
- Data generation using NumPy
- Conversion rate analysis
- Statistical hypothesis testing (t-test)
- Visualization using Matplotlib

## Results
- Variant B showed higher conversion rate (~20%)
- Variant A conversion ~15%
- Statistically significant difference (p-value < 0.05)

## Key Insights
- Identified higher-performing variant
- Improved conversion by ~15–20%
- Demonstrated A/B testing workflow

## Tools Used
Python, Pandas, NumPy, Matplotlib, SciPy
